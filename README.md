# 李琪 Liki personal website

这是一个纯静态个人艺术家网站，可以直接打开 `index.html` 预览，也可以部署到 GitHub Pages、Netlify、Vercel 或任意静态空间。

## 修改位置

- 页面内容：`index.html`
- 作品总览：`works.html`
- 个人信息：`about.html`
- 作品详情：`pages/`
- 视觉样式：`styles.css`
- 语言切换与滚动显现：`script.js`
- 网页图片：`assets/`

## 常改内容

- 邮箱：当前为 `lnanjingee@gmail.com`，如需修改可在所有 HTML 文件中搜索替换。
- 作品描述：在每个 `<article class="work">` 内修改中文 `.zh` 和英文 `.en` 段落。
- 作品图片：把新图片放进 `assets/`，再修改对应的 `src="assets/..."`。
- 多图画廊：重点作品的横向图组在 `assets/galleries/`。

原始作品资料没有被移动或改写，网页只使用了压缩后的副本。
